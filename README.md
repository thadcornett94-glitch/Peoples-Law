## 🎯 Overview

People's Law is a comprehensive Legal Knowledge and Civics Platform designed to make the legal system transparent and accessible. Empower citizens with information about their rights, responsibilities, and how to navigate legal systems with confidence.

## 🚀 Getting Started

### Prerequisites
- Node.js 14+
- npm or yarn
- Gemini API key
- Modern web browser

### Installation

```bash
# Clone the repository
git clone https://github.com/thadcornett94-glitch/Peoples-Law.git
cd Peoples-Law

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Add GEMINI_API_KEY and database credentials

# Start development server
npm run dev

# Access at http://localhost:3000
```
